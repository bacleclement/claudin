# Claudin

Free, open-source LinkedIn outreach powered by Claude Code + Chrome extension.
Replace Waalaxy, Lemlist, and Phantombuster with AI.

## Skills

| Skill | Purpose |
|---|---|
| /claudin-setup | Interactive wizard — business context, targets, messages, schedule |
| /claudin | Daily pipeline — search, connect, message on acceptance |
| /claudin-dashboard | Pipeline stats — visual + JSON |

## Prerequisites

- Claude Code desktop app (not CLI)
- Claude in Chrome extension installed and connected
- Chrome open with LinkedIn logged in

## Files

| File | Committed | Purpose |
|---|---|---|
| config.example.json | Yes | Template for user config |
| config.json | No (gitignored) | User's config — generated by /claudin-setup |
| tracker.json | No (gitignored) | Contact database — auto-managed |
| reports/ | No (gitignored) | Daily reports (JSON + Markdown) |
